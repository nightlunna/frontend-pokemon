# Pokedex API & Exibição Pokémons

Um projeto Full-Stack para gerenciamento e visualização de Pokémons, contando com uma API, integração com banco de dados SQL e uma interface web para visualização dos Pokémons adicionados.

## Funcionalidades

* **Listagem de Pokémon:** Consome a API interna para exibir os Pokémon salvos.
* **Visualização de Status:** Exibe nível, HP, ataque, defesa, ataque especial, defesa especial e velocidade.
* **Gerenciamento:** Adição e remoção de itens/Pokémon (via API).
* **Persistência de Dados:** Conectado a um banco de dados SQL para salvar todas as informações permanentemente.

## Ferramentas Utilizadas

* **Front-end:** Vite, React, CSS
* **Back-end:** Python, Flask (https://github.com/nightlunna/api-pokemon.git)
* **Banco de Dados:** SQL (Gerenciado via SQL Admin)
